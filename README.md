# Readthechart & Ccolor – Vision Education Demo

A simple HTML/CSS/JavaScript single-page app for vision education, designed for educational purposes only.

## ⚠️ Important Disclaimer

**This website is for educational and demonstration purposes only. It is not a medical diagnostic tool and should not be used to diagnose, treat, or assess any vision conditions. Always consult with a qualified eye care professional for proper vision testing and diagnosis.**

## Features

- **Readthechart**: Educational reading demo with 35 original sentences and timing functionality
- **Contrast**: Demonstrates high, medium, and low contrast text
- **Ccolor**: Custom color-vision demo with dot patterns
- **Astigmatism**: Radial line pattern demonstration
- **Depth**: Educational content about depth perception

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling
- `script.js` - All interactive functionality

## Local Development

Simply open `index.html` in a modern web browser. No build tools or server required.

## GitHub Pages Deployment

The code is already committed locally. To deploy:

### Option 1: Using GitHub Website (Easiest)

1. **Create the repository on GitHub:**
   - Go to https://github.com/new
   - Repository name: `vision-screening-pro`
   - Make it **Public** (required for free GitHub Pages)
   - Click "Create repository"

2. **Push your code:**
   ```bash
   git push -u origin main
   ```

3. **Enable GitHub Pages:**
   - Go to your repository: https://github.com/alisabam/vision-screening-pro
   - Click **Settings** → **Pages**
   - Under **Source**, select **main** branch
   - Click **Save**
   - Your site will be live at: `https://alisabam.github.io/vision-screening-pro/`

### Option 2: Using the Deploy Script

```bash
chmod +x deploy.sh
./deploy.sh
```

Then follow the prompts to create the repository if it doesn't exist.
